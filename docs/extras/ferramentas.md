# Ferramentas

> **📝 Página em construção.** Liste aqui ferramentas úteis para os estudantes.

## Simuladores

- **FluidSIM** — circuitos pneumáticos e eletropneumáticos *(Festo)*
- **CADe SIMU** — simulador de comandos elétricos
- **LogiSim Evolution** — circuitos digitais
- **Multisim / Proteus** — circuitos eletrônicos

## Software de programação

- **Python** — análise de dados, automação, scripts
- **MATLAB / Octave** — cálculo numérico e simulações
- **TIA Portal** — programação de CLPs Siemens
- **CODESYS** — programação de CLPs (norma IEC 61131-3)

## Ambientes para drones e VANTs

- **Mission Planner** — estação de controle terrestre para ArduPilot
- **QGroundControl** — estação de controle multiplataforma
- **Gazebo** — simulação de robótica
- **PX4 / ArduPilot** — firmwares de controle de voo

## Produtividade acadêmica

- **Zotero** — gerenciamento de referências bibliográficas
- **Overleaf** — editor LaTeX colaborativo online
- **Obsidian** — notas pessoais com vínculos
- **Mermaid** — diagramas em texto

## Plataformas educacionais

- **Khan Academy** — fundamentos de matemática e física
- **Tinkercad** — simulação de circuitos para iniciantes
- **MIT OpenCourseWare** — material acadêmico aberto
